# 🎣 Ловец-Съёмка — радар заказов на GitHub Actions

Мониторит YouDo и Profi.ru, отбирает несложные съёмки с бюджетом ~20–50 тыс ₽
и шлёт их в Telegram. **Учится**: под каждым заказом кнопки 👍/👎 — твои оценки
попадают в промпт Claude, и следующие заказы фильтруются с учётом твоего вкуса.

Крутится целиком на GitHub Actions — компьютер держать включённым не нужно.
Запуск каждые ~10 минут с 8:00 до 23:00 МСК (GitHub может задерживать на 3–10 мин).

## Отличия от локального Ловца

| | Ловец (локальный) | Ловец-Съёмка (GitHub) |
|---|---|---|
| Где работает | твой Mac, всегда включён | серверы GitHub |
| Скорость реакции | ~1 мин | ~10–20 мин |
| Нейросеть | LM Studio локально | Claude API (~100–300 ₽/мес) |
| Profi.ru | твой Chrome-профиль | куки в секрете, протухают ~раз в 1–4 недели |
| Обучение | нет | да, кнопки 👍/👎 |

## Секреты (Settings → Secrets and variables → Actions)

| Секрет | Что это | Где взять |
|---|---|---|
| `TG_BOT_TOKEN` | токен твоего бота | @BotFather → /newbot |
| `TG_CHAT_ID` | твой Telegram ID | @userinfobot |
| `ANTHROPIC_API_KEY` | ключ Claude API | console.anthropic.com → API Keys |
| `PROFI_COOKIES` | куки Profi (опционально) | см. ниже |

⚠️ После создания бота напиши ему любое сообщение (нажми Start) — иначе он не сможет тебе писать.

## Куки Profi.ru

1. Поставь расширение **Cookie-Editor** в Chrome.
2. Залогинься на profi.ru в свой кабинет специалиста.
3. Cookie-Editor → Export → JSON (скопируется в буфер).
4. Вставь JSON в секрет `PROFI_COOKIES`.
5. Когда куки протухнут, бот напишет в Telegram — повтори шаги 2–4.

Не хочешь возиться — поставь `enabled = false` в секции `[profi]` файла `lovec/config.toml`.

## Настройка фильтров

Всё в `lovec/config.toml` — правится прямо на GitHub (открыть файл → карандаш → Commit).
Бюджетная вилка, ключевые слова, стоп-слова, радиус, часы работы, порог оценки Claude.

## Как учить бота

- 👍 «Моё» — таких заказов будет больше.
- 👎 «Мусор» — таких будет меньше.
- Оценки хранятся в `data/feedback.json`; последние 40 подмешиваются в промпт Claude.
- Реакции подхватываются при следующем запуске (до ~10–20 мин).

## Важно про репозиторий

- Делай репозиторий **публичным**: у публичных Actions бесплатны без лимита.
  У приватных лимит 2000 мин/мес — при запуске каждые 10 минут его не хватит.
  В коде и конфиге секретов нет, токены живут в GitHub Secrets.
- Первый запуск — вручную: Actions → Lovec → Run workflow.
- Расписание на публичном репо отключается после 60 дней без коммитов,
  но бот сам коммитит состояние каждый запуск, так что это не грозит.

## Почему нет Avito

У Авито нет ленты заказов от клиентов (это витрина исполнителей), плюс жёсткий
антибот против серверных IP. Если появится источник — добавить адаптер в
`lovec/sources/` и подключить в `run.py` по образцу `youdo.py`.
